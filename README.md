# StreamIt
Live-streaming application with RTMP and WHIP protocols integrated
StreamIt is a full-stack web app providing a platform for creators to share real-time video content.
It has been developed using the T3 stack; NextJs for optimized React components, Typescript for logic and functionality, Tailwind for styling and customization of our UI and LiveKit for our live streaming functionality. 
LiveKit is a cloud Web Real-Time Communication platform allowing for real-time content delivery such as real-time chat functionality and streaming.
LiveKit allows us to provide real-time interactivity to users of our Web app by chatting with their followers as they stream as well as follow and participate in fellow creator streams.
Account management and authentication is undertaken using ClerkJs, a third party authentication library integrated into our app using their NextJs SDK and WebHooks.
